Some basic GIT commands

pwd	prints the working directory to the screen
cat	prints out the contents of a file
clear	clears the screen
ls	list the contents of the current directory to the screen
ls -la	(list all) lists even more details
cd	changes directory
cd ../	goes up one level
cd /
cd ~	goes to home directory
mkdir	adds a new directory (folder)
rm	removes a file
rm -rf	removes a directory and everything in it
